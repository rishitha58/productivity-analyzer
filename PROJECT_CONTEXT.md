# PRODUCTIVITY ANALYZER PROJECT

## TECH STACK
- Frontend: React + Vite + Tailwind CSS v3
- Backend: Node.js + Express + MongoDB
- NLP Service: Python + FastAPI + spaCy
- AI: Grok AI API
- Maps: Google Maps API
- Auth: JWT
- State: Zustand
- Animations: Framer Motion
- Icons: Lucide React

## CORE FEATURES
1. User signup asks: Student? Long-term goal? Duration?
2. Daily journal in natural language → converted to to-do list
3. Dashboard shows tasks (lighter color), tick when done
4. Sleep schedule daily, alarms, min 5 hour sleep rule
5. AI recommendations (Grok)
6. Study Mode: beautiful UI, music icon, AI icon, DnD with calls allowed
7. Sidebar: Important Points, Notes, Mistakes, Mock Tests
8. AI generates important points from topics or finds missing points in PDFs
9. Mistakes tracked from mock tests → review section
10. Mock tests built from AI study conversations
11. Travel: extract location, calculate time, notify before leaving
12. Food recommendation based on time and user preference
13. Habit drift detection
14. Long-term goal → phases → milestones → daily tasks
